#  RiskVerse AI – Invest Without Fear

>  AI-powered platform to learn investing without risking real money

---

##  Overview

RiskVerse AI is an AI-powered investment simulator that helps Gen-Z users overcome fear of investing.

Practice with virtual money, understand risk, explore IPOs, and get simple AI-driven insights.

 Learn → Experiment → Build Confidence → Invest in Real Life

---

## 🎯 Key Features

-  Virtual Investment Simulator (₹10,000)
-  Risk Intelligence Dashboard
-  Loss Probability Meter
-  AI Portfolio Explainer
-  Personality-Based Investing
-  Future Self Simulator
-  Smart IPO Advisor
-  Gamified Learning System

---

##  Tech Stack

- Frontend: React.js
- Backend: Node.js / Express
- AI: OpenAI API
- Database: MongoDB
